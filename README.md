大家好，这是我第一次上传我在GitHub的项目，首先我得要介绍一下这个程序是什么。

这个程序是用Intellij Idea所做，以MyBatis+Spring+SpringMVC（也就是人们所说的SSM框架）
搭建而成。

它的内容目前是一个最简单的前后台，其前台是显示暗黑破坏神2的各个符文情况
（全部为33个，这里因为懒，所以才放置了14个）。后台则是通过输入框的提示，
添加新的信息（符文）到数据库，再通过前台将MySQL储存的符文信息显示出来。

以后如果我有时间，我会将这些代码进一步完善，预期内可能会把33个符文全部补完，
并且还会新增符文之语的内容。
